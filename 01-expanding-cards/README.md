# 01 — Expanding Cards

**[Live Demo](https://lyuboss.github.io/50-projects-in-50-days/01-expanding-cards/)**
· Part of [50 Projects in 50 Days](../README.md)

## What it does

Click a card to expand it. The previously active card shrinks back automatically.

## Key concepts practiced

- `querySelectorAll` + `forEach` to attach events to multiple elements
- `classList.add` / `classList.remove` to manage active state
- A helper function (`removeActiveClasses`) to reset all panels before setting a new one
- CSS `flex` and `transition` for the smooth expand/shrink animation
