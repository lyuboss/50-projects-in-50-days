  # 03 — Rotating Navigation

**[Live Demo](https://lyuboss.github.io/50-projects-in-50-days/03-rotating-navigation/)**
· Part of [50 Projects in 50 Days](../README.md)

## What it does

Click the menu button to rotate the entire main content page, revealing a hidden side navigation menu. Click the close button to rotate the page back to its original position.

## Key concepts practiced

- CSS `transform: rotate()` and `transform-origin` to pivot the entire page layout
- CSS transitions to create smooth animations and stagger the entrance of list items
- Positioning (`absolute` and `fixed`) to place the circular buttons and off-canvas menu
- JavaScript DOM manipulation using `classList.add()` and `classList.remove()` to manage the active state on the main container
