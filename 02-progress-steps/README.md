# 02 — Progress Steps

**[Live Demo](https://lyuboss.github.io/50-projects-in-50-days/02-progress-steps/)**
· Part of [50 Projects in 50 Days](../README.md)

## What it does

Click next/prev buttons to move through a 4-step progress bar.
The progress line fills and empties based on the current step.

## Key concepts practiced

- DOM manipulation to update active classes on multiple elements
- Button disabled state management (first and last step)
- CSS custom properties to control progress bar width dynamically
